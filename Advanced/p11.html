CSS Pseduo-elements...

The css pseudo-elements allow you to sytle the elements or parts of the elements without adding any IDs or class to them

CSS3 introduce a new double-colon(::) syntax for pseudo-elements to distinguish between them and pseudo-classes.

selector::pseudo-element{property:value;}
