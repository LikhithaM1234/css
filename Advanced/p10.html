CSS PSUEDO-CLASSES...

the css pseduo-classes allow you to style the dynamic states of an elements such as hover, active and focus state, as well as elements that are existing in the document tree but can't be targeted via the use of other selector without adding any IDs or classes to them ,for example targeting the first or last child elements

A pseudo- class starts with a colon (:).Its syntax can be given with :

selectors:pesudo-class{property:value;}
